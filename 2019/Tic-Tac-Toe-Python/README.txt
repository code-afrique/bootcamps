Tic Tac Toe Python created by Jesse Phillips on December 19, 2018

Citation: All files in the ‘game2d’ folder in the project directory were created by Professor Walker White of Cornell University.

Instructions:
Run ‘python Tic-Tac-Toe-Python’ in the directory containing the ‘Tic-Tac-Toe-Python’ folder


Player 1 uses the arrow keys to move, space bar to place down a token
Player 2 uses the WASD keys to move, and E to place down a token
